# Data Processing (SQL)

## Deskripsi

Folder ini berisi jawaban SQL untuk seluruh soal pada bagian Data Processing.

Seluruh query disimpan pada file

```
answer.sql
```

---

## Daftar Soal

### 1. Insert Data

Menambahkan data employee baru:

- Name : Albert
- Position : Engineer
- Join Date : 24 Januari 2024
- Experience : 2.5 Years
- Salary : $50

---

### 2. Update Data

Mengubah salary seluruh employee dengan posisi Engineer menjadi

```
$85
```

---

### 3. Aggregate

Menghitung total pengeluaran salary pada tahun 2021.

---

### 4. Sorting

Menampilkan tiga employee dengan Years of Experience tertinggi.

---

### 5. Subquery

Menampilkan employee dengan posisi Engineer yang memiliki experience kurang dari atau sama dengan 3 tahun menggunakan subquery.

---

## Menjalankan Query

Buka database menggunakan MySQL atau PostgreSQL, kemudian jalankan seluruh isi file

```text
answer.sql
```

atau

```sql
SOURCE answer.sql;
```

---

## Hasil

Seluruh query dapat dijalankan secara independen sesuai kebutuhan masing-masing soal.
